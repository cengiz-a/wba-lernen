Warum reveal.js?

  • einfache und moderne Präsentationsentwicklung mit HTML
  • interaktive und leichte Präsentationsmethode
  • nach eigenen kreativen Vorstellungen
  • kostenlos und erweiterbar
    • reveal.js: Ein Framework zum erstellen schöner, moderne Präsentationsfolien mit HTML/Markdown
    • es basiert auf JavaScript
    • lässt sich durch jeden modernen Browser darstellen
    • auch offline
    • kostenlos, erweiterbar, Code verfügbar, mehr Aufmerksamkeit

Präsentation erstellen (vereinfacht)

  1. Einbettung des css-Frameworkdokumentes und des Themes im 
    <head>

    <link rel="stylesheet" href="../revealjs-framework/css/reveal.min.css">
<link rel="stylesheet" href="../revealjs-framework/css/theme/medieninformatik.css" id="theme">
  2. Einbettung des JavaScript-Codes im 
    <body>

    <script src="../revealjs-framework/lib/js/head.min.js"></script>
<script src="../revealjs-framework/js/reveal.min.js"></script>
  3. Konfigurieren von reveal.js unter der JavaScript-Einbettung
    <script>
    Reveal.initialize({
    controls: true,
    progress: true,
    history: true,
    center: true,

    transition: 'slide', // none/fade/slide/convex/concave/zoom
</script>
Der Kern von reveal.js kein PDF, kein HTML-Tag-Erkenner,...

Workshoppraxis 1

Gestaltet das Grundgerüst einer Präsentation in Aufgabe 1. Nutzt dazu reveal.min.css und das Medieninformatik-Theme medieninformatik.css. Damit die Präsentation funktioniert müsst ihr außerdem den JavaScript-Code in euer Dokument einbinden.

Außerdem könnt ihr euch einen Folienübergang aussuchen.

  • augabe_1.html findet ihr im Ordner revealjs-basics-training
  • Die reveal-Dateien sowie das Theme sind im Ordner revealjs-framework
    Beachtet, dass ihr zum einbinden zunächst einen Ordner hoch müsst (../).

Content einfügen

  • Content kann normal per HTML dargestellt werden
  • Gestaltung des Contents geschieht mittels CSS
  • Content Strukturierung mittels Markdown ebenfalls möglich
    <section data-markdown>
    <script type="text/template">
        ## H2-Überschrift

        Ein Paragraph mit Link [link](http://medieninformatik.th-koeln.de).
    </script>
</section>
  • Css primär aus reveal.min.css und des Themes unregelmäßig anpassungen mit style=""
  • sEinige css-Klassen sagen reveal wie es zu interagieren hat: Bsp: fragment
  • Markdown-Vorteil: Schnelle "Editierung", Nachteil: Man muss sich drangewöhnen/lernen
  • kein vollständiger HTML Ersatz
