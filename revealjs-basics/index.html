Warum reveal.js?

  • einfache und moderne Präsentationsentwicklung mit HTML
  • interaktive und leichte Präsentationsmethode
  • nach eigenen kreativen Vorstellungen
  • kostenlos und erweiterbar
    • reveal.js: Ein Framework zum erstellen schöner, moderne Präsentationsfolien mit HTML/Markdown
    • es basiert auf JavaScript
    • lässt sich durch jeden modernen Browser darstellen
    • auch offline
    • kostenlos, erweiterbar, Code verfügbar, mehr Aufmerksamkeit
