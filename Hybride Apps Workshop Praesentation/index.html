Was sind Hybride Apps?

  • mit HTML5, CSS3 und Javascript programmierte Webapps die in einer Art Browser auf einem Smartphone laufen
  • können für verschiedene Plattformen gepublished werden

Was sind die Vorteile?

Hybride Apps - Vorteile

  • es muss kein nativer Code geschrieben werden
  • können komplett offline verfügbar gemacht werden
  • der Zugriff auf Hardwarekomponenten ist möglich

Was sind die Nachteile?

Hybride Apps - Nachteile

  • es kann zu Problemen mit den Zugriffsrechten kommen
  • einschränkungen bei plattformspezifische Interaktionen
  • Performanceschwächen bei rechenintensiven Anwendungen

Übersicht Frameworks

  • Ionic ist einfach zu schreiben und ermöglicht dank Cordova umfangreiche Hardwareintegration
  • Intel XDK es werden Bootstrap 3, jQuery mobile, Ionic und das Topcoat-UI-Framework unterstüzt.
  • Onsen UI ist dank jQuery und vieler fertiger Komponenten besonders Einsteigerfreundlich
  • Famous legt den Focus auf 2D- und 3D-Rederings und ist somit ideal für Spielentwicklung

Hardwareintegration ng-Cordova Kamera

  1. HTML 
      
<img ng-show="imgURI !== undefined" ng-click="takePicture()"
    ng-src="{{imgURI}}"   class="circular">
<img ng-show="imgURI === undefined" ng-click="takePicture()"
    ng-src="img/placeholder.png" class="circular">
  2. Javascript
    $scope.takePicture = function () {
        var options = {
            quality: 75,
            destinationType: Camera.DestinationType.DATA_URL,
            sourceType: Camera.PictureSourceType.CAMERA,
            allowEdit: true,
            encodingType: Camera.EncodingType.JPEG,
            targetWidth: 300,
            targetHeight: 300,
            popoverOptions: CameraPopoverOptions,
            saveToPhotoAlbum: false,
            correctOrientation:true
        };
        $cordovaCamera.getPicture(options).then(function (imageData) {
            $scope.imgURI = "data:image/jpeg;base64," + imageData;
        }, function (err) {
            // An error occured. Show a message to the user
        });
};
