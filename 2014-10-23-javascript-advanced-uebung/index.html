Entwickeln Sie ein JavaScript, dass die eingegebenen Daten des Kontaktformulars validiert.
  • Nachname muss ausgefüllt sein
  • Nachricht muss mehr als 20 Zeichen enthalten
  • E-Mail-Adresse muss syntaktisch korrekt sein
  • Bei invaliden Datum soll das entsprechende Feld rot gefärbt werden
  • Zusätzlich soll in dem dazugehörigen Label ein Hinweistext erscheinen
  • Erst wenn alle Voraussetzungen erfüllt sind, soll das Formular abgesendet werden
  • Das JavaScript soll in einer externen JS-Datei entwickelt werden und in der formular.html eingebunden werden
