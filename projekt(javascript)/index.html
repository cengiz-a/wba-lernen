Willkommen zu unserem kleinen Tutorial zu AngularJS.

Hier lernst du die ersten Schritte zu AngularJS und einige coolen Funktionen dazu.
Wir hoffen sehr, dass du Spaß daran haben wirst, mit uns ein wenig über das Framework AngularJS zu lernen und wir sind immer offen für Feedback sowie weiteren Fragen.
Natürlich benötigst du Grundkentnisse in HTML, CSS und JavaScript, da das Framework auf JavaScript aufbaut und man ohne HTML oder CSS keine Internetseiten im Browser anzeigen lassen kann.
Starte das Tutorial in dem du in der Navigation auf "Los geht's" drückst und starte beim ersten Kapitel oder bei einem Kapitel nach deiner Wahl.
