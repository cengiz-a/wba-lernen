Kapitel 1 - AngularJS einbinden in unsere HTML Datei

Schön, dass du dich für unser Tutorial entschieden hast. Um mit AngularJS zu beginnen, benötigt ihr zuerst das Framework. Ihr könnt euch aussuchen, ob ihr es downloaden oder über einen Link einbinden wollt. Im folgenden Tutorial binden wir das Framework über einen Link ein und haben es nicht lokal auf unserem Rechner.

Code: 
<!DOCTYPE html>
<html lang="de">
  <head>
    <title>Unsere erste Webapp</title>
  </head>
  <body ng-app>
    <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.2.11/angular.min.js"></script>
  </body>
</html>
An dieser Stelle passiert noch nichts mit deiner Internetseite. Wir haben erstmal das Framework AngularJS mit deiner Internetseite verknüpft. JavaScript-Dateien fügt man standardmäßig am Besten immer am Ende des body-Bereiches ein, da so die Seite schneller laden kann. Zusätzlich müsst ihr euren body-Tag mit einem "ng-app" versehen, damit das Framework AngularJS erkennt, dass es sich hierbei um eine AngularJS App handelt.

Kapitel 2 - Zwei-Wege-Datenbindung

Wir beginnen mit einem simplen, jedoch sehr faszinierenden Beispiel. Mit AngularJS ist es möglich eine Zwei-Wege-Datenbindung zu erstellen, womit sich der Text automatisch aktualisiert.

Code: 
<!DOCTYPE html>
<html lang="de">
  <head>
    <title>Unsere erste Webapp</title>
  </head>
  <body ng-app>
    Guten Tag {{.deinName }}<br />
    <input type="text" ng-model="deinName" placeholder="Name hier eingeben" />
    <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.2.11/angular.min.js"></script>
  </body>
</html>
Mit AngularJS ist es auch möglich mathematische Rechnungen durchzuführen, die auch dank der Zwei-Wege-Datenbindung automatisch live aktualisiert werden. Wer braucht bei solchen tollen Features noch einen Taschenrechner, wenn man es auch selber machen kann?

Code: 
<!DOCTYPE html>
<html lang="de">
  <head>
    <title>Unsere erste Webapp</title>
  </head>
  <body ng-app>
    <input type="number" ng-model="zahl1" placeholder="Zahl 1 hier eingeben" />
    <input type="number" ng-model="zahl2" placeholder="Zahl 2 hier eingeben" /><br />
    Produkt: {{.zahl1 }} * {{.zahl2 }} = {{.zahl1 * zahl2 }}<br />
    Summe: {{.zahl1 }} + {{.zahl2 }} = {{.zahl1 + zahl2 }}<br />
    Division: {{.zahl1 }} / {{.zahl2 }} = {{.zahl1 / zahl2 }}<br />
    Subtraktion: {{.zahl1 }} - {{.zahl2 }} = {{.zahl1 - zahl2 }}<br />
    <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.2.11/angular.min.js"></script>
  </body>
</html>
Beispiel:
Guten Tag {{deinNameTut}}


Wir ergänzen <input> mit der Direktiven ng-model, somit verweist der Inhalt des Inputs auf die Bezeichnung deinName. AngularJS erkennt dies und aktualisiert das DOM somit automatisch.

Beispiel 2:

Produkt: {{zahl1}} * {{zahl2}} = {{zahl1 * zahl2}}
Summe: {{zahl1}} + {{zahl2}} = {{zahl1 + zahl2}}
Division: {{zahl1}} / {{zahl2}} = {{zahl1 / zahl2}}
Subtraktion: {{zahl1}} - {{zahl2}} = {{zahl1 - zahl2}}

Man erkennt, dass man innerhalb der {} Klammern auch Rechnungen durchführen kann. AngularJS erkennt diese Rechnung und gibt letztendlich das Ergebnis aus. Anders als bei anderen Programmiersprachen, muss man hier nicht bestimmen, ob es sich um den Typen double, float, int oder andere Datentypen handeln soll. In JavaScript gibt es keine Typisierung von Variablen.

Kapitel 3 - Der Controller

Nun kommen wir zum Controller. Wir ergänzen unsere HTML Tags mit der Direktive ng-controller und können somit Funktionen programmieren und ausführen, die wir in einer seperaten JavaScript-Datei abspeichern. Nicht vergessen am Ende des Body-Tags die app.js einzubinden.

Code: index.html 
<!DOCTYPE html>
<html lang="de">
  <head>
    <title>Unsere erste Webapp</title>
  </head>
  <body ng-app>
    <div ng-controller="hauptController">
      {{.hello_world }}
    <div>
    <script src="js/app.js"></script>
    <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.2.11/angular.min.js"></script>
  </body>
</html>

Code: app.js 
function hauptController($scope) {
  $scope.hello_world = "Hallo Welt!";
}
Ein weiteres Beispiel ist die ng-repeat Direktive, in der wir ein Array auslesen können und so jedes Element anzeigen lassen können. Es verhält sich ähnlich einer for-each-Schleife, die man aus anderen Programmiersprachen kennen könnte.

Code: index.html 
<!DOCTYPE html>
<html lang="de">
  <head>
    <title>Unsere erste Webapp</title>
  </head>
  <body ng-app>
    <div ng-controller="hauptController">
      <h3>Studentenliste:</h3>
        <ul>
          <li ng-repeat="student in studenten">{{.student.name }}</li>
        </ul>
    <div>
    <script src="js/app.js"></script>
    <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.2.11/angular.min.js"></script>
  </body>
</html>

Code: app.js 
function hauptController($scope) {
  $scope.studenten = [
    {name: "Nikolas Beckel"},
    {name: "Jacquelyn Rappenhöner"},
    {name: "Peter Parker"}
  ];
}
Eine weitere Funktion des Controllers ist, dass wir eigene Eingaben in unser vorhandes Array hinzufügen können. Wir erzeugen einfach ein neues input-Element und ein neuen button. In das input-Element kommt die Direktive ng-model hinein, welches den Inhalt in die jeweilige Variable hinzufügt. ng-click wird dem button hinzugefügt, welches dafür sorgt, dass durch das Klicken auf den Button die Funktion neuerStudent() ausgeführt wird. Zusätzlich bauen wir noch eine Suchfunktion ein, in der wir dank der Filterfunktion von AngularJS mit Hilfe von filter : SUCHVARIABLE nach bestimmten Buchstaben/Wörtern suchen kann.

Code: index.html 
<!DOCTYPE html>
<html lang="de">
  <head>
    <title>Unsere erste Webapp</title>
  </head>
  <body ng-app>
    <div ng-controller="hauptController">
      <h3>Studentenliste:</h3>
        <input type="text" ng-model="inpStudentSuche" placeholder="Suche..." />
        <ul>
          <li ng-repeat="student in studenten | filter : inpStudentSuche">{{.student.name }}</li>
        </ul>
        <input type="text" ng-model="inpStudent" placeholder="Student hunzufügen" />
        <button type="submit" ng-click="neuerStudent()">Hinzufügen</button>
    <div>
    <script src="js/app.js"></script>
    <script src="https://ajax.googleapis.com/ajax/libs/angularjs/1.2.11/angular.min.js"></script>
  </body>
</html>

Code: app.js 
function hauptController($scope) {
  $scope.hello_world = "Hallo Welt!";

  $scope.studenten = [
    {name: "Jacquelyn Rappenhöner"},
    {name: "Nikolas Beckel"},
    {name: "Peter Parker"},
    {name: "Johannes Reiter"},
    {name: "Marie Marien"}
  ];

  $scope.neuerStudent = function() {
    $scope.studenten2.push({
      name: $scope.inpStudent
    });
    // Textfeld leeren
    $scope.inpStudent = '';
  };
}
Beispiel:
{{hello_world}}

Studentenliste:

Hier kann man erkennen, dass für jedes Element in studenten ein neues li-Element erzeugt wird und bekommen somit eine Liste aller Studenten ausgegeben.
Beispiel 3

Die Filterfunktion ist ganz einfach hinzuzufügen. Man fügt einfach zu der Direktive ng-repeat | filter: inpStudentSuche hinzu und schon hat man eine simple Suchfunktion entwickelt.
In unserer app.js fügen wir eine neue Funktion neuerStudent() hinzu, in den wir unsere Eingabe rein pushen, was man auch aus anderen Programmierensprachen kennen könnte. Das pushen fügt den eingegebenen Inhalt auf das Array hinzu.

Kapitel 4 - Die Module

Eine weitere tolle Funktion von AngularJS sind die Module. Sobald eine Internetseite immer mehr an Inhalt und Funktionen bekommt, wird sie schnell unübersichtlich. Dafür sind die Module gedacht, die den Code strukturieren und leserlicher gestalten.
Der Codebeginn unseres Controllers

Code: app.js 
function hauptController($scope) {
  // Code
}
verändert sich wie folgt:

Code: app.js 
var hauptApp = angular.module('myApp', []);

hauptApp.controller('hauptController', function($scope) {
 // Code
});
Nun liegt es an dir!
Finde nun selbstständig heraus, wofür die Module gut sein könnten und inwiefern sie dir die Arbeit erleichtern. AngularJS bietet und liefert noch (un)endlich viele Möglichkeiten deine Internetseite zu gestalten.
