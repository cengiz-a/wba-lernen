3D Transformationen Beispiel

Bei diesem Beispiel werden 6 verschiedene HTML Elemente (figure) mit Grafiken gefüllt und mit 3D Transformation verschoben.
Zuerst wird die Grafik in CSS mit dem Attribut "background" gesetzt und danach mit den Operationen rotateX/rotateY und translateZ transformiert.
Um nun die verschiedenen Seiten anzuschauen, kann man über die Knöpfe mit jQuery eine Klasse setzen. Wenn die Rückseite angezeigt werden soll, kann im Element mit der ID "box" also die Klasse auf "show-back" gesetzt werden. Damit sich das Element nun auch dreht, muss in CSS eine Transition (= Übergang) gesetzt werden. Diese Transition sollte im Hauptelement mit der ID "box" gesetzt werden.
