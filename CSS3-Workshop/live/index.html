Animation Properties

  • animation-name
  • animation-duration
  • animation-timing-function (linear | ease)
  • animation-delay
  • animation-iteration-count (n | infinite )
  • animation-direction (normal | reverse | alternate)
  • animation-fill-mode
  • animation-play-stat (paused | running)

JQuery vs. CSS3


  • CSS3 Animationen bestehen aus einem Stil, über den die Animation beschrieben wird, und einem Set von Keyframes

  • CSS Animationen Performanter, da vom Browser aus gesteuert wird

  • nicht jeder Browser unterstützt CSS3

  • Einzelne Animationen bei CSS3 nicht möglich - Bsp. rotation und scale unabhängig voneinander zu verschiedenen Zeiten

    • JQuery vs. CSS3


  • JQuery muss eingebunden werden

  • Genauere Animationen sind mit JQuery einfacher umzusetzen

  • Bei CSS3 nur von Keyframe zu Keyframe anhaltbar

  • Bei JQuery kann man bei jeder Animationssequenz stoppen

  • CSS3 für einfache Animationen wie :hover oder mouseover animationen ideal

  • Mit Jquery sind einzelne Animationsschritte aufrufbar
