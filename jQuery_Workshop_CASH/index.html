jQuery

  • JavaScript-Code bzw. Bibliothek
  • reduziert den Codeaufwand von JavaScript-Anwendungen erheblich
  • einfache Funktionen ermöglichen Erstellung von Animation
  • es existieren mehrere tausend Funktionen
  • Einbettung in HTML:
  • Code direkt im script Tag innerhalb des head oder head Tags
  • Code verlinkt im script Tag innerhalb des body Tags

Syntax

jQuery Syntax ist darauf zugeschnitten HTML-Elemente zu finden und Aktionen an ihnen auszuführen

$(selector).action()

  • $ signalisiert Aufruf von jQuery
  • (selector) wählt HTML-Elemente aus
  • action() ist der auszuführende Befehl

Das Dokument muss vollständig geladen sein, bevor die jQuery-Funktionen ausgeführt werden. Andernfalls können Fehler auftreten.

jQuery Events

  • jQuery click() - Der einfach Klick mit der Maus
  • jQuery hover() - Gleiten der Maus
  • jQuery mouseup() - loslassen der Maustaste
  • jQuery mousedown() - drücken der Maustaste

jQuery Selectors

  • $(“p’’).hide() - Auswahl eines HTML-Elements
  • $(“#test“).hide() - Auswahl einer ID
  • $(“.test“).hide() - Auswahl einer Klasse
  • $(“this“).hide() - Auswahl eines aktuellen HTML-Elements
